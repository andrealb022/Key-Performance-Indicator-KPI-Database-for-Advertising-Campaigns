# Key-Performance-Indicator-KPI-Database-for-Advertising-Campaigns

The system will store information about the companies, their campaigns, and the ads shown during sports events. Key aspects include tracking campaign details, advertisements, products, and user engagement metrics such as impressions, reach, and sales.
